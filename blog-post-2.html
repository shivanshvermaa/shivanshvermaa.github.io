CAPTCHA Detection and Breaking using Neural Networks

By Shivansh Verma

A CAPTCHA (an acronym for "completely automated public Turing test to tell computers and humans apart") is a type of challenge–response test used in computing to determine whether or not the user is human. CAPTCHA, as the explanation of the acronym implies, determines whether an online request or submission is made by a human or a computer. It generates and grades tests that humans can easily pass but are impossible for computer programs to solve – like having to type the scrambled or distorted words and letters inside a picture (or voice recording for the visually impaired).

The above is the sample cleaned image using an OpenCV 2x2 kernel grid and using an Erode function to remove all spaces that qualify.


This relatively cleaned image was used for trainging after manually identifying all characters and building up a dataset of characters of such kind.


As YOLOv3 Model for object detection was selected due to it's lightweight and relative ease of training we need to translate the Tensorflow code to an API that can understand the commands. For the same purposes we can use the Darkflow repository made by GitHub user thtrieu.

TL;DR

• Leveraged OpenCV to make the CAPTCHA more machine readable using several Open CV functions and cleaned the image.
• From the cleaned images created a dataset with bounding boxes for training purposes.
• Utilized the YOLOv3 Model to create an object detection model based on the dataset prepared after preprocessing and cleaning.
• Model reached ~71% accuracy. One major limitation was that I did not have any GPUs for faster training purposes.
